# ===================================================================
# CHESS ENGINE REQUIREMENTS (EXTERNAL INSTALLATIONS)
# ===================================================================
# You must install Stockfish chess engine separately for AI gameplay:
#
# RECOMMENDED ENGINE:
# Stockfish (Free, Open Source, World's Strongest Engine)
#
# INSTALLATION BY PLATFORM:
#
# Windows:
#   - Download from: https://stockfishchess.org/download/
#   - Or using Chocolatey: choco install stockfish
#   - Or using Scoop: scoop install stockfish
#
# macOS:
#   - Using Homebrew: brew install stockfish
#   - Using MacPorts: sudo port install stockfish
#
# Linux (Ubuntu/Debian):
#   - sudo apt-get update
#   - sudo apt-get install stockfish
#
# Linux (CentOS/RHEL/Fedora):
#   - sudo yum install stockfish (CentOS/RHEL)
#   - sudo dnf install stockfish (Fedora)
#
# Linux (Arch):
#   - sudo pacman -S stockfish
#
# ALTERNATIVE ENGINES (Optional):
# Any UCI-compatible engine will work:
# - Komodo (Commercial): https://komodochess.com/
# - Leela Chess Zero (Neural Network): https://lczero.org/
# - Dragon (Commercial): https://shop.komododragon.com/
#
# ENGINE AUTO-DETECTION:
# The Flask app automatically searches for Stockfish in these locations:
# - System PATH
# - stockfish.exe (Windows)
# - /usr/local/bin/stockfish
# - /usr/bin/stockfish
# - /opt/homebrew/bin/stockfish (macOS Homebrew)

# ===================================================================
# SYSTEM REQUIREMENTS
# ===================================================================
# - Python 3.7 or higher
# - Operating System: Windows 10+, macOS 10.14+, or Linux
# - Memory: Minimum 1GB RAM (2GB+ recommended for engine battles)
# - Disk Space: ~100MB for application + dependencies + engine
# - Web Browser: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
# - Network: Local network access for web interface (localhost:5000)

# ===================================================================
# INSTALLATION INSTRUCTIONS
# ===================================================================
# 1. Clone the repository:
#    git clone https://github.com/top-site/chess3
#    cd chess3
#
# 2. Create virtual environment (recommended):
#    python -m venv venv
#    source venv/bin/activate  # Linux/macOS
#    venv\Scripts\activate     # Windows
#
# 3. Install Python dependencies:
#    pip install -r requirements.txt
#
# 4. Install Stockfish engine (see ENGINE REQUIREMENTS above)
#
# 5. Optional - Create .env file for configuration:
#    SECRET_KEY=your-secret-key-here
#    FLASK_ENV=development
#
# 6. Run the application:
#    python app.py
#
# 7. Open web browser and navigate to:
#    http://localhost:5000

# ===================================================================
# DEPLOYMENT CONFIGURATIONS
# ===================================================================
# For Production Deployment:
# - Uncomment gunicorn or waitress above
# - Set FLASK_ENV=production in environment
# - Use a proper secret key (not the default)
# - Consider using a reverse proxy (nginx, Apache)
#
# For Docker Deployment:
# - Ensure Stockfish is included in container
# - Expose port 5000
# - Set appropriate environment variables
#
# For Heroku Deployment:
# - Add Stockfish buildpack
# - Configure Procfile with web server
# - Set environment variables in Heroku dashboard

# ===================================================================
# TROUBLESHOOTING
# ===================================================================
# If you encounter issues:
#
# 1. Engine Not Found:
#    - Verify Stockfish installation: `stockfish` in terminal
#    - Check application logs for engine detection errors
#    - Ensure engine has execute permissions on Unix systems
#
# 2. Port Already in Use:
#    - Change port in app.py: app.run(port=5001)
#    - Kill existing Flask processes
#
# 3. Permission Errors:
#    - Run with appropriate user permissions
#    - Check file system permissions for temp directories
#
# 4. Browser Compatibility:
#    - Update to supported browser version
#    - Enable JavaScript
#    - Clear browser cache and cookies
